# Launchpad

## Launchpad Token Creation

- **Create Launchpad**
    - **Verify Token**: Ensures that the token to be launched is legitimate and verified for security and trustworthiness.
    - **DeFi Launchpad Info**: Collects essential information about the DeFi project, such as project description, goals, and financial details.
    - **Additional Info**: Provides extra details about the launchpad, enhancing transparency and information for potential investors.
    - **Finish Page**: Final review and confirmation page to finalize the launchpad setup.

- **Create Fair Launch**
    - **Verify Token**: Confirms the authenticity of the token to be launched.
    - **DeFi Launchpad Info**: Gathers detailed information about the DeFi project.
    - **Additional Info**: Adds supplementary information to give investors a complete understanding.
    - **Finish Page**: Concludes the setup process with a final review.

- **Create Dutch Auction**
    - **Verify Token**: Ensures the token is authentic and ready for auction.
    - **DeFi Launchpad Info**: Collects necessary data about the project.
    - **Additional Info**: Additional details to provide comprehensive project information.
    - **Finish Page**: Final step to review and confirm the auction setup.

- **Create Subscription**
    - **Verify Token**: Verifies the token for the subscription sale.
    - **DeFi Launchpad Info**: Gathers all required project details.
    - **Additional Info**: Extra information to support investor decision-making.
    - **Finish Page**: Final review and confirmation of the subscription setup.

- **Create Token**
    - **Standard Token**: Allows creation of a standard ERC-20 token.
    - **Liquidity Generate Token**: Creates tokens with built-in liquidity generation mechanisms.
    - **Buyback Baby Token**: Special token type with a buyback mechanism to support token price.
